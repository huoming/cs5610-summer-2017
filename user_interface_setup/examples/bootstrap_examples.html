Bootstrap Grid System

Use grid classes to layout page into rows and columns. Use row class to layout the page vertically. Use column family of classes to layout the page horizontally.

Twelve

There are 12 column classes that set a div's width to 12 different sizes. Column class names are cs-xx-1, cs-xx-2, etc, cs-xx-12.
